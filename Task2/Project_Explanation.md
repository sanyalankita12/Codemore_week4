# File-Based Notes Application

## Introduction

This project is a simple File-Based Notes Application developed using Python.

The application allows users to:

* Add notes
* View saved notes
* Delete notes
* Store notes permanently in a text file

## Technologies Used

* Python
* File Handling
* Functions
* Loops
* Conditional Statements
* Exception Handling

## Features

1. Add Note
2. View Notes
3. Delete Note
4. Exit Application

## Learning Outcomes

Through this project, I learned:

* Working with files in Python
* Creating reusable functions
* Handling user input
* Implementing menu-driven applications
* Organizing Python code effectively

## Conclusion

This project demonstrates the practical implementation of core Python concepts in a useful utility application.
